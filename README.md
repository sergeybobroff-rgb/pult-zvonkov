# Пульт контроля звонков — Android APK

## Как получить APK (бесплатно, без установки Android Studio)

### Шаг 1 — Создайте аккаунт на GitHub
Зайдите на [github.com](https://github.com) → Sign up (бесплатно)

### Шаг 2 — Создайте репозиторий
1. Нажмите кнопку **+** → **New repository**
2. Repository name: `pult-zvonkov`
3. Выберите **Private** (приватный, только для вас)
4. Нажмите **Create repository**

### Шаг 3 — Загрузите файлы
1. На странице репозитория нажмите **uploading an existing file**
2. Перетащите ВСЕ файлы из этой папки (включая `.github` и `www`)
3. Нажмите **Commit changes**

⚠️ Важно: папка `.github` скрытая, убедитесь что она включена в загрузку!

### Шаг 4 — Дождитесь сборки
GitHub автоматически запустит сборку APK.
- Перейдите во вкладку **Actions** вашего репозитория
- Дождитесь зелёной галочки ✓ (обычно 5–10 минут)

### Шаг 5 — Скачайте APK
1. В Actions нажмите на последний workflow run
2. Внизу страницы найдите **Artifacts**
3. Скачайте **Пульт_звонков_APK**
4. Распакуйте ZIP → внутри `app-debug.apk`

### Шаг 6 — Установите на телефон
1. Перешлите APK на телефон (через Telegram, email, USB)
2. Откройте файл на телефоне
3. Android спросит разрешение на установку из неизвестных источников → Разрешить
4. Установить → Готово!

При первом запуске приложение попросит доступ к микрофону → Разрешить.

## Что умеет приложение

- Слушает звуки через микрофон телефона
- Распознаёт звонки 4 телефонных линий: AT1, MSH, GDU, A2
- Отображает красную лампочку и счётчик звонков
- Хранит историю за 7 дней
- Кулдаун 40 сек независимо по каждой линии
- Настройка чувствительности, переименование линий
